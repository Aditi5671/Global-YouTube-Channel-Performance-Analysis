# Global Youtube Channel Performance Analysis 📊

## Overview

This project analyzes a dataset of YouTube channels to identify trends in channel performance, categories, and countries. The analysis was performed using Python libraries to explore patterns related to subscribers, views, uploads, and estimated earnings.

## Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab / Jupyter Notebook

## Analysis Performed

* Identified **top countries** with the most YouTube channels
* Analyzed **most popular channel categories**
* Created a **correlation heatmap** to study relationships between numerical features
* Performed **exploratory data analysis (EDA)** and visualized key trends

## Key Insights

* Channels with higher **video views tend to have more subscribers**
* **Entertainment and Music** are among the most common channel categories
* Monthly views show a relationship with **estimated yearly earnings**
